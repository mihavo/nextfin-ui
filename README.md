## Getting Started

### Prerequisites

- Node.js >= 18
- pnpm (preferred package manager)

### Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/your-username/nextfin-ui.git
   cd nextfin-ui
   ```

2. Install dependencies:

   ```sh
   pnpm install
   ```

3. Start the development server:

   ```sh
   pnpm dev
   ```
